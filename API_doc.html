class Frontend.pages.Page_reviews.page_review(dataframe: DataFrame, folder_path: Path)[source]

The class page_review manages the Streamlit page for analyzing recipe reviews.

This class facilitates an interactive dashboard that enables users to explore: - The temporal distribution of comments (reviews) for recipes. - The relationship between the number of comments and recipe usage across years. - Seasonal associations of recipes based on the timing of their reviews.

It leverages Streamlit functionalities to provide visualizations such as: - An annual histogram of reviews. - Pie charts showing seasonal distributions. - Interactive filters for recipes based on review counts and popularity.
